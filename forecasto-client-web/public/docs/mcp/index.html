Overview

Panoramica

Forecasto is a financial management platform designed for small businesses, freelancers, and consultants. It helps you track cashflow, manage invoices, monitor bank accounts, and calculate VAT obligations.

The Forecasto MCP connector lets you interact with your Forecasto workspace directly from Claude. You can create and update financial records, generate cashflow projections, manage bank accounts, and handle VAT calculations — all through natural conversation.

Forecasto è una piattaforma di gestione finanziaria progettata per piccole imprese, liberi professionisti e consulenti. Ti aiuta a monitorare il cashflow, gestire fatture, conti bancari e calcoli IVA.

Il connettore MCP di Forecasto ti consente di interagire con il tuo workspace direttamente da Claude. Puoi creare e aggiornare record finanziari, generare proiezioni di cashflow, gestire conti bancari e calcolare l'IVA — tutto attraverso una conversazione naturale.

Getting Started

Come Iniziare

1. Connect on Claude.ai
Go to Settings → Integrations in Claude.ai and search for Forecasto in the connector directory. Click Connect and authorize with your Forecasto account via OAuth 2.0.

1. Collegati su Claude.ai
Vai su Impostazioni → Integrazioni in Claude.ai e cerca Forecasto nella directory dei connettori. Clicca Connetti e autorizza con il tuo account Forecasto tramite OAuth 2.0.

2. Or connect via Claude Code
Add the Forecasto MCP server to your Claude Code configuration:

2. Oppure collegati tramite Claude Code
Aggiungi il server MCP di Forecasto alla tua configurazione Claude Code:

// settings.json or .mcp.json { "mcpServers": { "forecasto": { "type": "url", "url": "https://app.forecasto.it/mcp" } } }

3. Start asking
Once connected, simply ask Claude about your finances. The connector automatically determines which tools to use.

3. Inizia a chiedere
Una volta collegato, chiedi a Claude delle tue finanze. Il connettore determina automaticamente quali strumenti usare.

Authentication

Autenticazione

Forecasto uses OAuth 2.0 Authorization Code flow to securely connect your account. When you connect for the first time, you'll be redirected to Forecasto's login page to grant access. Tokens are automatically refreshed — no manual intervention needed.

Forecasto usa il flusso OAuth 2.0 Authorization Code per collegare il tuo account in modo sicuro. Al primo collegamento, verrai reindirizzato alla pagina di login di Forecasto per autorizzare l'accesso. I token vengono aggiornati automaticamente — nessun intervento manuale necessario.

MCP Server Endpoint
Endpoint Server MCP

https://app.forecasto.it/mcp

The connector requests access only to the workspaces and data associated with your Forecasto account. You can revoke access at any time from your Forecasto account settings.

Il connettore richiede accesso solo ai workspace e ai dati associati al tuo account Forecasto. Puoi revocare l'accesso in qualsiasi momento dalle impostazioni del tuo account.

Usage Examples

Esempi d'Uso

Example 1 — Cashflow Projection
Esempio 1 — Proiezione Cashflow
“Show me the cashflow forecast for the next 3 months, grouped by week.”
“Mostrami la proiezione del cashflow per i prossimi 3 mesi, raggruppata per settimana.”
Uses get_cashflow to retrieve inflows, outflows, net cashflow, and running balance for the specified period.
Usa get_cashflow per recuperare entrate, uscite, cashflow netto e saldo progressivo per il periodo specificato.
Example 2 — Create an Invoice Record
Esempio 2 — Creare un Record Fattura
“Create a new invoice for Acme Corp, €5,000 + VAT 22%, due date June 15th, in the orders area.”
“Crea una nuova fattura per Acme Corp, 5.000€ + IVA 22%, scadenza 15 giugno, nell'area ordini.”
Uses create_record with area "orders", calculating VAT amount automatically and setting the cashflow date.
Usa create_record con area "orders", calcolando automaticamente l'importo IVA e impostando la data di cashflow.
Example 3 — VAT Position Check
Esempio 3 — Verifica Posizione IVA
“What’s my VAT position for Q1? Show me a dry run of the quarterly calculation.”
“Qual è la mia posizione IVA per il primo trimestre? Mostrami un dry run del calcolo trimestrale.”
Uses calculate_vat with dry_run=true to preview IVA a debito vs. IVA a credito without creating settlement records.
Usa calculate_vat con dry_run=true per visualizzare IVA a debito vs. IVA a credito senza creare record di liquidazione.
Example 4 — Multi-Workspace Analysis

“Compare the cashflow of my two companies for this quarter and show me a consolidated view.”

Usa list_workspaces e get_consolidated_cashflow per combinare i dati finanziari di più entità.

Data & Privacy

Dati e Privacy

The Forecasto MCP connector accesses only the financial data in your Forecasto account, strictly scoped to the workspaces you have permission to access. Data is transmitted over HTTPS/TLS and is only shared with Claude when you explicitly request it through conversation.

Il connettore MCP di Forecasto accede solo ai dati finanziari del tuo account, limitatamente ai workspace a cui hai accesso. I dati sono trasmessi via HTTPS/TLS e vengono condivisi con Claude solo quando lo richiedi esplicitamente nella conversazione.

What we DO NOT collect
Cosa NON raccogliamo

Forecasto does not store credit card numbers, bank routing numbers, or passwords. We do not sell data to third parties or use your data for AI model training.

Forecasto non memorizza numeri di carte di credito, coordinate bancarie o password. Non vendiamo dati a terzi né utilizziamo i tuoi dati per l'addestramento di modelli AI.

For full details, see our Privacy Policy.

Per tutti i dettagli, consulta la nostra Privacy Policy.
